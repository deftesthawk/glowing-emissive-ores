    Instructions

    There are currently 4 Minecraft Modloaders: NeoForge, Forge, Fabric, and Quilt. You will need to install different requirements depending on which Modloader you are using. I have listed all the possible combinations you can use for each Modloader in the requirements tab.

    Confused? Here's an example:

    I plan on playing a Forge modpack: First, I install "Glowing Emissive Ores", then test if it works (some packs come pre-installed with the requirements). If it doesn't work, then come back to this site and select the requirements tab. Next, I will choose either Option 1, 2, 3, or 4 depending on the version of Minecraft I am playing on. If I choose option 1, I will need to install both "MoreMCMeta (Forge)" and "MoreMCMeta Emissive Textures (Forge)". Finally, I will load the modpack up again and test it. For most modpacks, this should work.

    Troubleshooting

    • If the modpack crashes ‚Äî try launching it at least three times, as large modpacks are sometimes prone to random crashes during startup.
    • If the ores aren‚Äôt glowing or the modpack continues to crash ‚Äî try a different option from the requirements list until you have tried all possible options. (If you've got Sinytra Connector installed, the only options that will work are fabric mods like Continuity)
    • If you're using shaders and the ores only glow at certain times ‚Äî look for a shader setting called ‚ÄúFloodfill‚Äù and disable it. Alternatively, use a different emissive ores resource pack that supports "LabPBR."
    • If the ores are flickering (a Z-fighting issue caused by pre-0.6 versions of Sodium conflicting with Continuity) - Try switching to a different requirements option (the Fusion option doesn't have this issue when combined with sodium).
    • If you're still experiencing issues ‚Äî feel free to leave a comment on my CurseForge page or join my Discord for quicker support.
